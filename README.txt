Fruits Database

Developed a database which stores reviews of different fruits.

Motivation

Developed it while completing the course "The Complete 2022 Web Development Bootcamp". Aim was to learn Storing/Extracting data from MongoDB using the native node driver..

Tech/Framework used

NodeJS, ExpressJS, and MongoDB.

Installation

Install all the dependencies by typing the command "npm install" inside command line.
